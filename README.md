Student Marks Management System:

A Python console-based application designed to store, manage, and analyze student academic records. This project demonstrates the use of core Python concepts to perform data storage and basic analytical operations such as average calculation and identifying top-performing students.

Project Overview:

The Student Marks Management System allows users to efficiently manage student marks through a simple, menu-driven console interface. It enables adding, viewing, and analyzing student records using Python data structures and control flow logic.
This project is ideal for understanding Python fundamentals and implementing real-world logic in a structured manner.

Features:

-Add new student records
-Update existing student marks
-View individual and all student records
-Calculate average marks
-Identify top scorer(s)
-Menu-driven console interface

Technologies & Concepts Used:

Python
-Dictionaries
-Loops
-Conditional statements
-Basic data analysis logic

Project Type:

-Console-Based Application
-Core Python Project
-Academic / Beginner-Friendly Project
